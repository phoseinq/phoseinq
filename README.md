I build the unglamorous plumbing that keeps traffic moving — **bots, dashboards, proxies, and the glue between them.** Mostly **Python** and **Bash**, mostly self-hosted, always reproducible. If it routes packets or runs on a server at 3 a.m., it's probably mine.

### `//` What I'm into

- 🛰️  **Proxy & VPN infrastructure** — panels, tunnels, monitoring, the lot
- 🤖  **Telegram bots** that automate the boring parts (and a few of the hard ones)
- 📊  **Self-hosted dashboards** that watch servers, traffic and uptime so I don't have to
- 🧩  **Browser & desktop tooling** — small extensions that fix annoying things
- ✍️  Writing it all up on [**pvboy.dev**](https://pvboy.dev)
